# Not Totally Accurate [RegNet: Multimodal Sensor Registration Using Deep Neural Networks] PyTorch Implementation

Extrinsic Calibration of LiDAR Camera given an RGB image and a projected Depth image. Implemented for learning purposes. Based on [RegNet: Multimodal Sensor Registration Using Deep Neural Networks](https://arxiv.org/pdf/1707.03167.pdf). Still playing around with it. This isn't intended to be a faithful implementation.
